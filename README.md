# Zomato-Power-BI-Analysis-and-Visualization
This project analyzes restaurant performance, customer demand, revenue trends, and customer satisfaction using SQL and Power BI.  The objective was to transform raw restaurant and order data into actionable business insights through data cleaning, SQL analysis, KPI development, and interactive visualizations.

## Business Objectives

- Analyze restaurant distribution across cities
- Identify top revenue-generating restaurants
- Evaluate customer demand by city and area
- Understand factors affecting customer ratings
- Track revenue and average order value trends

- ## Tools & Technologies

- MySQL
- Power BI
- Power Query
- DAX
- Excel
